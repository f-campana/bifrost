# Bifrost

Bifrost is the schema layer for the Sanctum + Bifrost system.

It owns contracts, skills, routing rules, and validation logic. The repo is the
public operating layer, not the knowledge base.

Current Phase 0 contents:
- `contracts/design-pipeline-contract.md`
- `skills/design/`
- `system-spec.md`

Sanctum consumes Bifrost's schema. Bifrost does not store source knowledge.
